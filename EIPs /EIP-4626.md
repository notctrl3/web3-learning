## [EIP-2612](https://eips.ethereum.org/EIPS/eip-4626):Tokenized Vaults

### Motivation                            
 代币化金库的标准化将降低生息金库的集成成本，同时创建更一致和稳健的实现模式
### Specification
 - 底层「资产（asset）」是一种 ERC-20 代币。 vault 管理这类资产。
 - vault 为使用者发行「份额（shares）」代币，这份额本身也是 ERC-20。也就是说，用户拿到的是「代表金库份额／收益权的代币」
 - 基本功能，例如：
   - deposit / mint：存入资产，换取份额
   - withdraw / redeem：燒掉份額，取回對應價值的資產。
   - totalAssets：查 vault 目前管理的底层资产总额（含收益）
   - convertToShares / convertToAssets：根据当前 vault 的资产／份额比率，计算资产 ↔ 份额之间的兑换比（理想情况下的估算，不考虑费用，不考虑滑点）
   - previewDeposit / previewWithdraw / previewMint / previewRedeem：模拟操作 
### Security Considerations
#### Slippage
  没有滑点的考虑，对于预期外的交易无法回退
#### Malicious Implementations  
  永远不要信任未经审计的 Vault​, 接口符合 ≠ 安全
#### convertTo & preview
  用途完全不一致
