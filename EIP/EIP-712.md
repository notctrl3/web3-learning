## EIP-712

### Motivation                            
 提高链下签名使用，减少gas费和交易次数。结构化签名数据方便展示让用户理解
### Specification
 `Type Structured Data`: 将用户签名数据结构化，方便展示给用户 
 struct Permit {
    address owner;
    address spender;
    uint256 value;
    uint256 nonce;
    uint256 deadline;
}

 `Domain Separator`: 明确签名生效上下文
  {
  "name": "MyToken",
  "version": "1",
  "chainId": 1,
  "verifyingContract": "0xTokenAddress"
}
